Information We Collect

This website is a static informational site hosted on GitHub Pages. We do not directly collect personal information through forms, user registration, or other interactive data submission methods on this site itself.

The interactive eligibility guide tool processes information you provide locally in your browser. This information is not transmitted or stored by us.

Analytics

We may use privacy-respecting analytics (e.g., Plausible, Umami, or similar server-side analytics that do not use cookies or track personal data) to understand website traffic and improve our content. These tools help us count page views and identify popular content without collecting personal data from visitors.

GitHub Pages

This website is hosted on GitHub Pages. GitHub may collect information such as IP addresses and user agent strings for security and operational purposes. Please refer to the GitHub Privacy Statement for more details.

How We Use Information

Any information gathered through privacy-respecting analytics is used solely to improve the website's content and user experience. We do not sell or share this data with third parties for marketing purposes.

Data Security

As a static website, we minimize data handling. Information processed by the interactive tool remains within your browser. For data related to GitHub Pages hosting, please refer to GitHub's security measures.

External Links

This website contains links to external sites, such as OPM.gov and VA.gov. We are not responsible for the privacy practices or content of these other sites. We encourage you to read their privacy policies.

Policy Updates

We may update this privacy policy from time to time. Any changes will be posted on this page. This policy was last updated on December 7, 2023.

Contact Us

If you have any questions about this privacy policy, please contact us at feedback@veterans-info.github.io.
